# Raw Data Directory

This directory stores raw downloads fetched by the pipeline. Files are not tracked in Git.

## Contents (HDD_v1)

Expected subdirectories and files include:

- `BINDING_DB/BindingDB_All_202512_tsv.zip` (raw BindingDB archive)
- `BINDING_DB/BindingDB_All_202512_cleaned.csv` (cleaned BindingDB table)
- `LINCS/2020/compounds_raw.csv` (LINCS compound metadata)
- `JUMP_CP/cpg0016/JUMP_CP_compounds.csv` (JUMP-CP compound metadata)
- `ANNOTATION_DB/compound_details.jsonl` (AnnotationDB compound details)

## Notes

- These files are generated by Snakemake rules. You should not edit them manually.
- If you change versions or URLs in `config/pipeline.yaml`, re-run the pipeline to refresh these files.
- Keep `docs/data_sources.md` up to date with any version changes.
