# Results Directory

This directory stores the final outputs from the HDD_v1 pipeline. Files are not tracked in Git.

## Outputs

- `HDD_v1.RDS`: the Harmonized Drug Dataset Version 1 as a `MultiAssayExperiment`.

## Related artifacts

- QC report: `qc/hdd_quality_control.html` (rendered from `qc/hdd_quality_control.Rmd`).

## Notes

- Results are generated by running Snakemake from the repository root.
- If you regenerate outputs with different versions or parameters, update `docs/data_sources.md` and `docs/devnotes.md`.
